# modryn-boilerplate

Lean Next.js starter for Modryn prototypes. Driven by Claude Code, not Copilot.

Intentionally minimal: just the runtime, configs, a couple of utilities, and three UI
primitives. No payments, email, auth, analytics, database, or SEO machinery — add those
per project only when there's a real need. See `CLAUDE.md` for conventions and how to start.

---

Next.js · TypeScript · Tailwind CSS · Vercel
